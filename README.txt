Topics Blocks Module

This contains the following blocks:
- latest x articles in a given category
- top x articles in a given category
- a site stats block

###### Hacked By Adam Hennessy ######

- Added admin support for altering title character length
- Fixed bug in number of titles to display (now works in Xoops 2.0.5)
- Added more blocks (six topics blocks and one stats in total) to allow for more topics to be displayed
- Changed admin logo to suit xoops style
- Empty blocks no longer displayed

#####################################

Added support for WFSection 1.01
Removed Random Partner Block due to this functionality being present in the core.
Renamed 'articles' to 'news items', and called 'wfsections' articles.
Looks somewhat strange with less than 2 items in each module, due to use of plurals ('news items', etc), but I am assuming you are going to put some content in your site before making it public ;-)

----==== Requirement ====----

Xoops 2.* or Greater

----==== Installing ====----

1. Upload the folder inside the zip to the Xoops modules folder.
2. Login to your site as Admin, In the Admin area of xoops goto 'modules' and use the pull down menu dialog and select install. Then just use the blocks you require.
